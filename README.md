# Longest-common-substring
Longest common substring (LCS) of a string in non contiguous case by removing multiple adjacent characters
Also prints all possible substring


### Sample Input:
Size of each array:

8

6

Array 1 values: 
1 2 3 4 7 5 8 9

Array 2 values: 
1 2 4 5 7 8

### Sample Output:
Length of LCS: 5
Possible values: 
1 2 4 7 8 
1 2 4 5 8
